Monitor specific elements on a webpage and get notified when they change.

## Features
- **Selector Tracking**: Target any element using standard CSS selectors.
- **Change Detection**: Automatically notices when text or content within the element updates.
- **Visual Feedback**: Highlights the changes or provides notifications.

## How to Use
1. Go to the page you want to monitor.
2. Open Utility Hub and select **Monitor DOM**.
3. Enter the **CSS Selector** of the element (e.g., `#price` or `.status-badge`).
4. Click **Monitorar**.
5. The extension will notify you if the content of that element changes.

## Pro Tip
You can find CSS selectors by right-clicking an element on a page and selecting **Inspecionar (Inspect)**. The ID or Class name will be visible in the developer tools.
