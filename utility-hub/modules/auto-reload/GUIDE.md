Keep your tabs updated automatically with custom time intervals.

## Features
- **Custom Intervals**: Choose how often the page should refresh (Seconds or Minutes).
- **Hard Reload**: Force a refresh that bypasses the browser cache (useful for development).
- **Status Indicator**: An 'AR' badge appears on the tab icon when active.

## How to Use
1. Navigate to the page you want to monitor.
2. Open Utility Hub and select **Auto Reload**.
3. Set your interval (e.g., 30 Seconds).
4. Click **Iniciar** to begin.
5. Click **Parar** to stop the automatic updates.

## Use Cases
- Monitoring real-time dashboards.
- Keeping a web session from expiring.
- Checking for stock updates or new results.
